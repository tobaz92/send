# Contribuer à Send

## Signaler un bug

Ouvrez une issue avec :
- Ce que vous avez fait
- Ce que vous attendiez
- Ce qui s'est passé
- Votre version PHP et votre hébergeur si pertinent

## Proposer une modification

1. Forkez le repo
2. Créez une branche (`git checkout -b fix/mon-fix`)
3. Commitez vos changements
4. Ouvrez une Pull Request

## Conventions

- PHP 8.1+ strict (`declare(strict_types=1)`)
- Commentaires en français
- Pas de dépendance externe
- Indentation : 4 espaces
- Testez avec `php -l` avant de soumettre

## Sécurité

Si vous trouvez une faille de sécurité, ne créez **pas** d'issue publique. Contactez directement le mainteneur.
